# Quick & Fresh

Web εφαρμογή διαχείρισης με δύο ενότητες: **Product List** (καρτέλα προϊόντος + κοστολόγηση Cost) και **Αρχείο Επικοινωνίας**.

## Εκκίνηση

Χρειάζεται [Node.js](https://nodejs.org) (v18+) εγκατεστημένο στον υπολογιστή σου. Το frontend είναι γραμμένο σε React (JSX) και το backend σε Express.

Πρώτη φορά (εγκατάσταση + build του React frontend):

```
npm run setup
```

Μετά, κάθε φορά που θες να το τρέξεις:

```
npm start
```

Και άνοιξε στον browser: `http://localhost:3000`

### Development mode (live reload στο React κατά την επεξεργασία)

Σε ένα terminal: `npm start` (τρέχει το API στο :3000)
Σε άλλο terminal: `cd client && npm run dev` (τρέχει το React dev server στο :5173 με hot reload, προωθεί τα `/api` requests στο :3000)

## Δομή δεδομένων

Όλα τα δεδομένα αποθηκεύονται στο `data/db.json`. Οι εικόνες προϊόντων αποθηκεύονται στο φάκελο `uploads/`. Δεν χρειάζεται εξωτερική βάση δεδομένων — όλα τρέχουν τοπικά.

## Product List

- Καρτέλα **Product List**: βασικά στοιχεία είδους (κατηγορία, κωδικός, barcode, περιγραφές GR/EN, status, ενεργό στο μηχάνημα/καταστήματα, εικόνες).
- Καρτέλα **Cost**: κοστολόγηση με αυτόματο υπολογισμό:
  - Καθαρή τιμή = Τιμή Πώλησης ÷ (1 + ΦΠΑ%)
  - F.C.% = ΠΤΚ ÷ Καθαρή τιμή
  - Κέρδος = Καθαρή τιμή − ΠΤΚ
  - Μικτό = Καθαρή τιμή × Ποσότητα
  - Κέρδος SUM = Κέρδος × Ποσότητα
  - Ίδιοι τύποι εφαρμόζονται και στον πίνακα Stores (F.C. Store / F.C. Q&F) με βάση τις τιμές πώλησης ανά κατάστημα.

## Αρχείο Επικοινωνίας

Στοιχεία εταιρείας/επαφής, status, ενδιαφέρον, άτομα επαφής, ιστορικό επικοινωνίας (1η/2η τηλεφωνική επικοινωνία, αποστολή mail, επίσκεψη) και παρατηρήσεις.

## Σημειώσεις
